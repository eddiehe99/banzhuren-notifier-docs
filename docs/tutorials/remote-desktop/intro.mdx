# 介绍

学校电脑本来就有的功能，无需安装任何东西。教室能通过教室电脑远程连上办公室电脑。

## 功能

:::tip 最重要的功能

    能在教室使用办公室电脑。

:::

:::tip 最重要的使用场景

    在教室晚修值班的老师，能直接连上办公室电脑，进行备课等操作，无需自己带笔记本电脑去教室。

:::
