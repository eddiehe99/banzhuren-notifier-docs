# 介绍

本工具可部分实现 [希沃云班牌](https://www.seewo.com/article/detail/396) 的功能。

## 功能

- 班主任通知（记录在 Word 文档里）在课间自动在课室的电脑上展示
- 可选：自动抓取家长的留言到通知文档里，并随班主任的通知在课间自动展示
